# Sample Requirements Document
## Online Banking System

REQ-1: The system shall allow users to log in using their credentials.

REQ-2: The application should respond quickly to all user interactions.

REQ-3: The system shall process transactions in real-time and also batch 
process all transactions every night at midnight.

REQ-4: Users must be able to transfer money to other accounts. 
The transferred amount shall be deducted immediately from the sender's balance.

REQ-5: The system shall be secure and protect user data appropriately.
